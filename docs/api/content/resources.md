## List resources

List Amazon Web Services resources consumed by the Cumulus system.

```endpoint
GET /resources
```

#### Example request

```curl
$ curl https://cumulus.developmentseed.org/api/dev/resources --header 'Authorization: Basic Base64EncodedCredentials'
```
